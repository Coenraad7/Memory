Class MainGame This is the maingame class and contains most of the code used for the memorygame itself. Variables in Class Maingame public int Turn: keeps track of what players turn it is. only used in multiplayer and is given a value from Whostarts. int[,] cardproperties: Contains info of every picturebox. [x.y] x = picturebox number, y = pairnumber and visible(0 or 1). int[,] grid: contains a set ammount of values depending on the difficulty level. int[] scores: keeps track of every player score in the current game. int card1, card2: Contains pairnumbers from the last picture boxes that have been clicked. int picture1, picture2: remembers the cardproperties number for when they can be removed. int count: keeps track of the ammount of cards that have been played away. int timercount: this value helps decreasing the timers time and increases when all the players had their turn. PictureBox cardinfo1, cardinfo2: Contains the picturebox information from the last 2 picturebox clicks. double multiplier: keeps track of a multiplier for the scoring system in SINGLEPLAYER. Method MainGame default method Method MainGame_Load Runs code when the form loads. this depends on the Variables.Loadgame Method init Creator Richard: init initizalizes most of the game. It creates pictureboxes, empties textboxes and fills them accordingly. Mostely depends on difficulty, amountplayers. Method initspecial This method only runs when a continue game is selected. it makes the pictureboxes that have been played away invisible. Method Savegame Creator Richard/Johnny: This method loads all necessary variables to a .sav file Method Loadgame Creator Richard/Johnny: This method loads all necessary variables to a .sav file Method scramble Creator Richard: This method initializes the cardproperties array and fills it with randomized number variating per difficulty. every number will be created only twice and will act as a pair in the game. Method rotate Creator Richard: This method is called when a picture box is clicked. it acts as a rotation in the memory game because the picture will change. after the second click it will also call paircheck() to check the cards. if the 3th card is clicked to fast it will also call Undorotate(). Method undorotate Creator Richard: undo's the rotated cards after it is called. This results in that you are able to view the cards before they are invisible or back to the default background. Method paircheck Creator Richard: This method will compares the values of card1 and card2. if they have the same value it will call Keepscore(true) and add 2 to the count. it also checks if the game is finished or not. if the cards arent the same Keepscore(false) will be called. Method keepscore Creator Richard: This is a method in the MainGame class/form. It makes sure the players get a score when correct = true. if its not it will rotate to the next player or decreases Multiplier. Method highscore Creator Richard: This is a method in the MainGame class/form. When a game is finished it check if the current SINGLEPLAYER score is higher than the lowest highscore of that difficulty and places it in the correct spot and saves it to highscores.sav.
